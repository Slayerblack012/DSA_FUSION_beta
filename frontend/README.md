# DSA Fusion Frontend

Frontend client for DSA Fusion Final.
Built with Next.js and TypeScript.

## Requirements

- Node.js 20+
- npm 10+

## Run in Development

	npm install
	npm run dev

Default URL:

- http://localhost:3000

## Build and Start

	npm run build
	npm run start

## Lint

	npm run lint

## Main Stack

- Next.js 15
- React 19
- TypeScript
- Zustand for local state
- TanStack Query for async server state
- Framer Motion for animation
- Monaco Editor integration

## Notes

- Frontend is designed to work with backend API routes exposed by the FastAPI app.
- Keep API endpoint configuration aligned with backend base URL in app config.
